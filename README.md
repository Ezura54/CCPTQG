“临床药师理论试题智能生成器” 是一款基于前沿大语言模型（LLM）开发的智能命题系统。旨在解决临床药师带教老师在出题过程中面临的耗时久、考点覆盖不全、病例编写困难等痛点。

不同于传统的静态题库抽题模式，本系统能够根据您上传的专属知识库（PDF文档），结合您勾选的特定考点（知识点），实时创作出全新的、高质量的临床试题。

🛠️ 使用说明书

只需简单 4 步，即可生成一套高质量的专业试卷。

第一步：模型配置 🔑

在左侧设置栏的顶部，配置 AI 模型参数：

选择服务商：

推荐使用 Google Gemini（响应速度最快）。

或选择 Custom / SiliconFlow 使用国内 DeepSeek 模型。

输入 API Key：输入您对应的 API 密钥（以 sk- 开头）。

🛡️ 安全提示：API Key 仅存储在您的浏览器本地内存中，刷新页面即销毁，系统不会保存您的密钥。

第二步：导入知识库 📂

点击**“导入知识库”**区域，或将文件拖拽至此。

支持多选 PDF 文件（例如：《高血压药物治疗指南.pdf》、《抗菌药物临床应用指导原则.pdf》）。

系统会自动解析文件内容，解析成功后文件名旁会显示绿色对勾 ✅。

第三步：试卷结构设置 ⚙️

根据考试需求，调整试卷结构：

试卷标题：例如“2025年第一季度心血管科临床药师阶段考核”。

题量配置：输入各类题型的数量（建议初次尝试：A型 5题，案例 1个）。

第四步：选择考点 & 生成 🎓

在**“考点覆盖配置”**列表中，勾选本次考试需要考察的知识点。

例如：只考“（20）药物相互作用”和“（18）个体化给药方案的制定”。

点击底部的 “🪄 基于选定考点生成” 按钮。

系统将开始分析文档并命题，右侧预览区会实时显示生成的试卷内容。

第五步：导出与打印 📤

生成完成后，您可以使用顶部工具栏：

🖨️ 打印预览：调用浏览器打印功能，直接连接打印机或另存为 PDF 文件。

📝 导出 Word：将试卷内容下载为 .md 或 .doc 格式，便于进行二次编辑和排版。

❓ 常见问题 (FAQ)

Q: 生成一套试卷需要多长时间？

A: 通常在 30-90 秒之间。具体时间取决于您上传的文档大小以及您要求的题目数量。Gemini 模型通常比其他模型更快。

Q: AI 出的题准确吗？

A: 系统通过 Prompt Engineering（提示词工程）严格限制 AI 仅从您上传的文档中提取信息，极大降低了“胡编乱造”的概率。但作为严肃的医疗考核，我们强烈建议您在打印分发前，务必由专业带教老师人工审核一遍试题和答案。

Q: 为什么生成的试卷有时候没有答案？

A: 试题和答案是分开生成的，通常答案位于试卷的最底部（有红色虚线分割）。如果未显示答案，可能是因为生成的文字量超过了 API 的单次响应上限（Max Tokens）。建议适当减少单次生成的题目数量，分批次生成。

Q: 如何获取 API Key？

A: * Gemini: 访问 Google AI Studio 申请。

DeepSeek: 访问 DeepSeek 开放平台 申请。
